# ViT Adversarial Attacks (ISIC-2019)

Репозиторий содержит код и материалы эксперимента из статьи по оценке устойчивости Vision Transformer (ViT) к состязательным атакам на дерматоскопических изображениях набора данных **ISIC-2019**.

В работе сравниваются три подхода:
- **FGSM**
- **PGD**
- **Разработанный комбинированный метод** (градиентное возмущение + структурные искажения)

## Содержание репозитория

- notebooks/ # Jupyter-ноутбуки эксперимента и визуализаций

- figures/ # Готовые изображения (графики, примеры атак, тепловые карты) из статьи

- requirements.txt # Зависимости проекта

- README.md

## Используемые данные

В эксперименте применяется выборка **1000 сбалансированных изображений** из датасета **ISIC-2019** (8 классов по 125 изображений).

Официальный источник данных:
- ISIC-2019 Challenge Database

В репозиторий изображения не включены (требуется скачать отдельно).

---

## Модель

В качестве модели использовался Vision Transformer для классификации изображений кожных поражений (8 классов).  
Весовые коэффициенты загружаются из локального чекпоинта.

Пример пути к весам в ноутбуках:
C:\melanoma_attacks\checkpoint-12670

## Подготовка окружения

Рекомендуемая версия Python: **3.10+**

## Визуализации

Папка figures/ содержит готовые материалы, используемые в статье:

- сравнение Accuracy и F1-score до/после атак

- примеры оригинальных и атакованных изображений

- распределения метрик сходства MSE / PSNR / SSIM

- примеры пар изображений с численными значениями метрик

## Основные результаты (кратко)

Оценивание проводилось на 1000 изображениях ISIC-2019:

- Clean Accuracy: 0.9640

- FGSM Accuracy: 0.1220

- PGD Accuracy: 0.0400

- Авторская атака Accuracy: 0.1670

Метрики сходства (MSE / PSNR / SSIM) подтверждают малую заметность искажений при сохранении высокой эффективности атак.

## Повторяемость эксперимента

Для воспроизведения результатов необходимо:
1) Скачать ISIC-2019 и сформировать сбалансированную выборку (1000 изображений).
2) Указать пути к данным и чекпоинту модели в ноутбуках.
3) Запустить ноутбуки из папки notebooks/.

!!! Обратите внимание: точные значения метрик могут незначительно отличаться в зависимости от версии библиотек, GPU и настроек вычислений.
